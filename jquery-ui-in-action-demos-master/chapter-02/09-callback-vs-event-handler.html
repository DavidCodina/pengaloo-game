Callback vs. Event Handler:


	$("#button1")
		.wrap("<div class='text-center my-4 p-4'></div>")
		.on('click', function() {


			//The two ways of handling events in jQuery UI.
			//These example are equivalent.

			//Callback:
			$( "#dialog-1" ).dialog({
				title: "Dialog 1",
				create: function() {
					$( this ).css( "display", "block" );
					console.log("#dialog-1 was created!")
				}
			});

			//Event handler:
			$( "#dialog-2" )
				.on( "dialogcreate", function() {
					$( this ).css( "display", "block" );
					console.log("#dialog-2 was created!")
				})

				.dialog({
					title: "Dialog 2"
				});
		});


These divs were not initially dialogs. Therefore, they were set to display:none until .dialog() was applied to them (by clicking the button).
This technique is an alternative to the approach taken in 08, whereby the divs were created at the same time as dialog() was applied.
