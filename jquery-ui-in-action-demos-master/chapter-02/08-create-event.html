08-create-event.html

Create a <div>, then make it a dialiog, triggering the "dialogcreate" event.

This implementation is somewhat different (and better) than the original.


	$("#button1")
		//The button container (for centering) is added using jQuery...
		.wrap("<div class='text-center my-4 p-4'></div>")

		//on 'click' <div id='dialog'> is created immediately after the button.
		.on('click', function() {
			$("#button1").after("<div id='dialog'>Then the <code>dialogcreate</code> event was detected, and a message was logged to the console.</div>")


		//Now that <div id='dialog'> exists, it is turned into an actual dialog,
		//which then fires the dialogcreate event handler.
		$( "#dialog" )
			.on( "dialogcreate", function() { console.log( "Dialog was created!" ); })
			.dialog({
				title: "You just created me!"
				});
		});
