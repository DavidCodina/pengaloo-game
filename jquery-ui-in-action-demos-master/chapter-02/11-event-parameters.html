Event Parameters:

In this example, the event object and ui object are logged to the console upon the creation of the dialog.


	$("#button1")
		.wrap("<div class='text-center my-4 p-4'></div>")

		.on('click', function() {
			$( "#dialog" ).dialog({
				create: function( event, ui ) {
					console.log( event );
					console.log( ui );
				}
			});
		});


Open the console!!!
